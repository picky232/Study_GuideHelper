# 시연 시나리오 (발표용)

> 배포 URL: https://study-guide-helper-frontend.vercel.app

## 준비물

- 시연용 계정 (사전 회원가입 완료)
- 스마트폰 (PWA 설치 + 알림 권한 허용 상태)
- PC 브라우저 (Chrome)

## 시나리오 흐름 (약 5분)

### 1. 서비스 소개 (30초)

- "목표와 마감일만 입력하면 AI가 매일 무엇을 공부할지 자동으로 설계해주는 학습 도우미"
- PWA 설치형 웹앱 — 앱스토어 없이 홈 화면에 설치

### 2. 로그인 → 홈 대시보드 (30초)

- 로그인 후 홈 화면: 오늘의 태스크, 목표별 D-Day, 전체 달성률, AI 코칭 메시지
- 포인트: "오늘 할 일이 한눈에"

### 3. 목표 설정 + AI 계획 생성 (핵심, 2분)

1. "목표 관리" → 새 목표 추가
2. STEP 1: 과목 `정보처리기사`, 시험 종류 `자격증`, 유형 `필기+실기`
3. STEP 2: 마감일 60일 뒤, 하루 3시간
4. STEP 3: 완료 단원/약점 입력 (선택) → 계획 생성
5. **AI 생성 대기 (~30초)** — 로딩 애니메이션 동안 설명:
   - "Claude AI가 실제 출제 단원 기반으로 일자별 태스크를 분배"
   - "망각 곡선(1·3·7일) 복습 태스크 자동 삽입"
6. 미리보기 화면: 총 129개 태스크, 학습/복습 구분, 일자별 그룹
7. "이 계획으로 시작하기" → 홈에 반영 확인

### 4. 완료 체크 + 복습 사이클 (1분)

- 오늘 태스크 하나 "시작" → 타이머 → 완료 처리
- 달성률 실시간 갱신
- 캘린더 이동: 내일 날짜에 방금 완료한 태스크의 `[복습]` 태스크가 자동 생성된 것 확인
- 포인트: "완료하는 순간 망각 곡선 기반 복습 일정이 자동으로 잡힘"

### 5. 학습 피드백 (30초)

- 피드백 탭: 주간 달성률, 일별 그래프, AI 명언
- 포인트: "달성률에 따라 AI가 코칭 메시지 생성"

### 6. 푸시 알림 (30초)

- 스마트폰 알림바에 도착한 알림 보여주기:
  "오늘 미완료 학습이 N개 남았어요! 지금 시작해볼까요?"
- 매일 밤 9시 자동 발송 (Vercel Cron), 미완료 태스크 있는 사용자에게만

### 7. 마무리 (30초)

- 기술 스택 한 줄 요약: React PWA + Vercel Serverless + Supabase + Claude API + FCM
- 비용: 인프라 전액 무료 티어, Claude API만 소량 과금

## 시연 팁

- AI 계획 생성은 D-60 기준 약 30초 소요 — 이 시간에 아키텍처 설명
- 시연 직전 알림 수동 발송 가능:
  `curl -X POST https://study-guide-helper.vercel.app/api/notify/send -H "Authorization: Bearer $CRON_SECRET"`
- 계획 재생성해도 태스크 중복 안 됨 (기존 계획 삭제 후 재생성)

## 예상 질문 대비

| 질문 | 답변 |
|------|------|
| AI가 어떻게 실제 단원을 아는가? | Claude가 시험 종류·과목명 기반으로 실제 출제 단원 지식 활용, 프롬프트에서 "1단원 학습" 같은 추상 제목 금지 |
| 계획이 밀리면? | 미완료 태스크 기반 재조정 기능 (regenerate) |
| 보안은? | JWT 인증, API 키 전부 서버 전용, RLS, Cron 엔드포인트 시크릿 보호 |
| 비용 확장성? | 사용자당 Claude API 호출은 계획 생성 시에만 — Haiku 모델로 회당 수원 미만 |
